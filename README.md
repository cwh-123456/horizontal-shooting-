# horizontal-shooting-横向射击
一个仿照《python编程：从入门到实践》第一个项目《外星人入侵》制作的小游戏（如果我厚着脸皮仅仅称为“仿照”的话）

本项目相较于原书《外星人入侵》的主要区别：
- 玩家飞船可以上下左右自由移动
- 敌人会持续发射子弹，且子弹有不同种类
- 玩家飞船显示生命值
- 添加了登录与页面跳转
- 添加了难度设置

ps.其实本项目并没有完成，还有很多想法有待实现。但是鉴于本项目只是用于练习python与pygame基础知识，这些想法就不实现了，留给下一个项目。

又：本来想在这个项目里练习一下python的特殊方法，然后发现实在太牵强，于是烂尾了...
